# Контекст для бизнес-режимов

Проект: Разработка AI-ассистента для собеседований

Цель: Помочь кандидатам на собеседованиях с ответами на вопросы в реальном времени.

Основные функции:
- Транскрибация речи в текст (STT)
- Генерация подсказок на основе вопросов (LLM)
- Поддержка различных профилей собеседований
- Кэширование для быстрого ответа

Технологии:
- Python + FastAPI для бэкенда
- Electron для десктоп-приложения
- Whisper для STT
- Ollama для LLM
