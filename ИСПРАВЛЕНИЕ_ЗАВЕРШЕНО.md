## Что нужно сделать

1. **Полностью закройте приложение** (все окна)

2. **Откройте терминал** и перейдите в папку проекта:

   ```bash
   cd C:\Users\RODION\CascadeProjects\live-hints
   ```

3. **Запустите приложение**:

   ```bash
   npm start
   ```

## Как проверить что работает

1. В консоли при запуске увидите:

   ```
   STT-Dynamic: Auto-detected audio mode: loopback
   ```

2. Запустите транскрипцию (Start Recording)

3. Переключите звук с Монитора на Наушники

4. В консоли появится:

   ```
   [DeviceMonitor] Изменение устройства: "Monitor" -> "Наушники"
   [DynamicAudioCapture] Начало захвата с устройства: Наушники
   ```

5. Транскрипция продолжится без перерыва

## Если всё ещё не работает

1. Убедитесь, что вы перезапустили приложение (не F5, а полное закрытие и открытие)
2. Проверьте права доступа к микрофону в Windows Settings
3. Если нужно, переключите режим вручную в настройках приложения
